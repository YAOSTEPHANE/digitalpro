==================================================
COMMANDES À COLLER DIRECTEMENT SUR LE SERVEUR
==================================================

Vous êtes déjà connecté au serveur.
Exécutez ces commandes UNE PAR UNE :


📁 ÉTAPE 1 : Créer les dossiers
--------------------------------
mkdir -p /var/www
cd /var/www


📦 ÉTAPE 2 : Installer Node.js 18
----------------------------------
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash

export NVM_DIR="$HOME/.nvm"
[ -s "$NVM_DIR/nvm.sh" ] && . "$NVM_DIR/nvm.sh"

nvm install 18
nvm use 18

# Vérifier
node --version


📦 ÉTAPE 3 : Installer PM2
----------------------------
npm install -g pm2


📥 ÉTAPE 4 : Cloner le repository
---------------------------------
cd /var/www

# Si le repo est PUBLIC, cette commande fonctionnera :
git clone https://github.com/YAOSTEPHANE/digitalpro.git

# Sinon, rendez-le public sur GitHub d'abord


📦 ÉTAPE 5 : Installer les dépendances
---------------------------------------
cd /var/www/digitalpro
npm install


🏗️  ÉTAPE 6 : Builder le projet
---------------------------------
npm run build


🚀 ÉTAPE 7 : Démarrer avec PM2
---------------------------------
pm2 start npm --name "digitalpro" -- start
pm2 save


✅ VERIFICATION
---------------
pm2 status
pm2 logs digitalpro
