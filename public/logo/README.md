# Logo Digitalpro Solutions

## Instructions

Pour utiliser le nouveau logo 3D de Digitalpro Solutions :

1. Placez votre image de logo dans ce dossier avec le nom : `logo.png`
2. L'image sera automatiquement utilisée comme :
   - Logo dans la navbar
   - Logo dans le footer
   - Favicon du site
   - Icône de l'application (PWA)
   - Image Open Graph pour les réseaux sociaux

## Formats recommandés

- **Format** : PNG avec fond transparent ou blanc
- **Taille recommandée** : 
  - Logo principal : 1200x1200px (pour une qualité optimale)
  - Minimum : 512x512px
- **Nom du fichier** : `logo.png`

## Emplacement

L'image doit être placée dans : `/public/logo/logo.png`
